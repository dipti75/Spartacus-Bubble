## 🚀 Overview

**Spartacus Bubble** is a visual design prototype created in Figma that focuses on modern UI components, intuitive interactions, and a clean visual language.  
This repository serves as the central reference for designers, developers, and stakeholders to review and collaborate on the design flow.
